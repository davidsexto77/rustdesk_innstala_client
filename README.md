## Asistencia Técnica Oficial de Innstala

Esta es la herramienta oficial de asistencia técnica de **Innstala**. 
Diseñada para ofrecer una conexión segura, rápida y sencilla entre nuestros técnicos y nuestros clientes.

## ¿Cómo funciona?

1.  **Descarga** nuestra herramienta para tu sistema operativo.
2.  **Ábrela** (no requiere instalación).
3.  **Proporciona** el ID de 9 dígitos que aparece debajo del logo de Innstala a nuestro técnico.
